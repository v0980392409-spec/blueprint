Canonical guidance for the `@/wizard-modal-dialog` page layout family and its markdown entrypoints.

## Usage
- Load [`../_shared/page.modal-dialog.common.md`](../_shared/page.modal-dialog.common.md) first to align shared modal-dialog variables and slot guidance.
- Load `wizard-modal-dialog._common.md` next for the wizard-specific contract.
