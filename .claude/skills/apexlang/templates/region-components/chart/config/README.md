# Chart Configuration Templates

## Purpose
This directory is reserved for chart configuration modules that may be added under the chart template family.

## What is in this directory

- No chart configuration module files are currently present.
- The parent chart family lives in `templates/region-components/chart/`.

## How to use this directory

- Do not load this directory by default.
- Only add or load files here when a chart scenario needs reusable configuration modules that do not belong in the top-level chart variants or feature files.
- Use the parent chart README as the main guide for the chart family.

## Maintenance
- If configuration modules are added later, list them here and explain when to use them.
- If this directory remains intentionally empty long-term, keep that fact explicit so readers do not assume files are missing.
