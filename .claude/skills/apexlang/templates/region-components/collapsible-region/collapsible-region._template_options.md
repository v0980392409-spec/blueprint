# Collapsible Region Template Options

Use the listed `static_id` as the exact value to pass in the region's `templateOptions`.
Example: `templateOptions: [expanded, scroll-default]`
`#DEFAULT#` remains a standalone entry when used. Do not concatenate it with another token or replace the documented accepted value with the emitted CSS class string.
